# 🚀 Bunk Matrix – Smart Attendance Calculator

A futuristic, interactive attendance & bunk planning platform designed for college students.

🌐 **Live Demo:**
👉 https://bunk-matrix-armaan.netlify.app/
---

## ✨ Features

### 📊 Live Attendance Calculator

* Real-time attendance percentage
* Smart bunk calculation (75% rule)
* Instant updates without reload

---

### 🧠 Smart Status System

* “Safe hai bro 😎”
* “Thoda sambhal ke 😬”
* “Class leni padegi Bhai 😭”
* Dynamic UI glow based on status

---

### 🗓️ Smart Bunk Planner (Calendar)

* Real monthly calendar view
* Each day = 6 classes logic
* Click to mark:

  * Attended ✅
  * Bunked ❌
* Hover preview of attendance change
* Month navigation (← →)

---

### 🔮 Auto Prediction System

* Predict future attendance
* “If you bunk next X classes → new %”
* AI-style popup suggestions

---

### 🎨 Premium UI/UX

* Glassmorphism + Neon Design
* Smooth animations & micro-interactions
* Circular progress meter (SVG)
* Glitch & tech-style transitions

---

### 🧲 Interactive Features

* Custom cursor effects
* Hover animations & ripple effects
* Dynamic glow feedback
* “Danger Mode” (red pulse when attendance drops)

---

### 🔐 Signature Reveal (Easter Egg 😈)

* Hidden owner signature
* Cursor-based spotlight reveal
* Click to fully reveal creator identity

---

### 💾 Data Persistence

* Saves data using localStorage
* Auto restore on refresh

---

## ⚙️ Tech Stack

* HTML
* CSS (Glassmorphism + Animations)
* JavaScript (Logic + Interactions)

---

## 🎯 How It Works

1. Enter total & attended classes
2. System calculates attendance %
3. Use planner to simulate future classes
4. Apply planner → updates stats
5. Track & optimize your attendance
